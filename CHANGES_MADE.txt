THE CORE PROBLEM & SOLUTION:

PROBLEM: 
   import { feature } from 'bun:bundle';
   import { foo } from 'src/services/foo';
   
   At runtime Node.js tries to find packages named 'src' and 'bun:bundle'
   and fails with "Cannot find package 'src'"

SOLUTION:
   
   1. setup.js creates node_modules/bun/bundle.ts
      └─ Provides actual module for bun:bundle import
   
   2. fix-imports.js converts imports:
      from 'src/services/foo' → from './services/foo'
      from 'bun:bundle' → from './node_modules/bun/index.js'
      └─ Makes all imports relative, resolvable by Node.js
   
   3. run-build.js orchestrates the full process:
      setup → npm install → tsc → fix-imports → verify
      └─ Uses || true to handle TypeScript errors gracefully
