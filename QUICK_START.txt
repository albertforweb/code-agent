COMMON ISSUES AND FIXES:
========================

Issue: "node: command not found"
Fix: Install Node.js from nodejs.org

Issue: "npm: command not found"
Fix: Re-install Node.js (npm comes with it)

Issue: dist/main.js is empty or missing
Fix: Check build output for TypeScript errors
     Run: npm run clean && node run-build.js

Issue: "Cannot find package 'src'" at runtime
Fix: fix-imports.js didn't run or didn't work
     Check: grep "Post-build" test.log

Issue: test.log not created
Fix: Run tests manually and check console output
     Or run: node run-tests.js (creates clearer output)
